Core takeaways / 核心結論

1. 本集主軸不是單一個股，而是 AI 基礎建設的資產化：Meta 若把 GPU / 算力從純自用資產轉成可租賃資源，會同時回應折舊疑慮、壓低算力單價、並讓 NeoCloud 類型商業模式的「朦朧美」被更可計算的價格取代。 2. 半導體零組件的漲價故事仍在延伸：記憶體價格仍上行但漲幅可能收斂；被動元件多數漲價成功；功率元件 / MOSFET 與後段封裝被主持人放在「稼動率拉滿後獲利跳很快」框架下。 3. 導線架是本集最具體的供應鏈瓶頸：secondary note 與 primary report 都提到 10–20% 漲價、lead time 從 3–4 個月拉長到 7–8 個月；這是可追蹤的事實型線索，但仍需廠商法說或產業資料驗證。 4. 光通訊不是主持人高信心主線，比較像「聽眾服務」與資金輪動觀察：SiPh / InP（Narrow-and-Fast）偏弱，Wide-and-Slow / VCSEL / AOC / Micro LED 敘事偏強；要避免把技術路線討論直接翻成買賣結論。 5. Social Worker Daily 的捷敏、群聯、國巨紀錄必須嚴格標為筆記作者個人操作日記，不是主持人建議。
