主題地圖

1. 高位市場：漲多、麻木、但流動性風險升高

Primary report 將本集整理為：長期報酬與財富效果讓既有持股者對回檔較不敏感，市場「跌下去容易被買」。但這不等於風險消失；若融資、處置、交易制度或負面傳聞同時作用，流動性可能迅速惡化。

2. AI 供應鏈：傳聞敏感度上升

NVIDIA 散熱 TIM / BOM 變動被列為本集重要傳聞。這類資訊若屬實，可能影響台股零組件供應鏈；但目前本流程沒有官方證據，因此只能列為風險監控。

3. 成熟製程：真需求或提前拉貨？

聯電、世界先進、GlobalFoundries、環球晶被整理為成熟製程 / 半導體上游材料轉強代表。關鍵不是短期股價，而是判斷拉貨是否來自真實終端需求，或只是對漲價、關稅、斷供的提前反應。

4. 被動元件：低基期補漲與短缺想像

MLCC、鋁電容、電感與日美被動元件標的被整理為族群性起漲。若 lead time、book-to-bill、報價與庫存同步改善，才可能升級為正式投資研究。

追蹤計畫 / event triggers

  • 成熟製程升級為正式研究：UMC / VIS / GFS 同步給出 utilization 上升、ASP 改善，且不是單季 pull-in。
  • 成熟製程降級為短線輪動：Q2 後月營收轉弱、客戶 forecast 未上修、庫存天數上升。
  • 被動元件升級：lead time 拉長、報價上調、book-to-bill > 1，且多家公司財報同時確認。
  • AI 散熱鏈風險解除：NVIDIA 或 Tier-1 供應鏈確認 TIM/BOM 傳聞影響有限，且相關股成交量與價格穩定。
