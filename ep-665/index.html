核心結論

  1. EP665 的主軸是高檔市場裡的訊號辨識。主持人承認盤勢「噴到看不懂」、帶有賭的成分，但仍把被動元件週期、SpaceX IPO / Neo Cloud、機器人與台積電分紅拆成不同可信度的研究線索。
  2. 被動元件是本集最可量化的主線。來源提到國巨 B/B ratio 拉升、特規品稼動率約 90%、標準品約 80%，並用 2018 年毛利率高峰作為壓力測試；真正要追的是漲價斜率與 EPS 上修，而不是單純追股價。
  3. SpaceX 是事件驅動 + 願景驅動。SEC S-1 / S-1A 讓 SpaceX 上市程序成為官方文件可追蹤事件；但軌道 AI server / Neo Cloud 仍是長期願景，不能直接當成已驗證商業模型。
  4. 二階題材要降級處理。太陽能、散熱、6/12 出貨日、機器人與個人操作日記多半來自筆記作者延伸或題材擴散，不能寫成主持人核心觀點。
  5. 台積電分紅目前更像情緒雜訊。除非後續出現薪酬制度或人才流動的硬資料，否則不應把員工抱怨升級為基本面看空。
