Core takeaways / 核心結論

1. 本集最核心的投資線索不是「全面追高」，而是市場從被動元件熱潮外溢到離散元件、PMIC、Power / 功率元件後，哪些題材仍有基本面支撐、哪些已轉為籌碼與情緒主導。

2. 被動元件被主持人描述為從「半信半疑」走到「全面相信」：基本面可能仍有肉，但短線技術面與籌碼面重要性上升，應避免把產業邏輯直接等同於可追價空間。

3. 台積電被放在「外面題材修正後資金回流避風港」的位置；這是資金流與配置上限的觀察，不是單純因台積電當下基本面新增利多。

4. 離散元件/功率元件的劇本偏向「高階需求排擠 + non-China 供應鏈配置 + 一線 IDM 轉往高階」造成台廠二供機會；但主持人也保留「不會超級屌缺」與「能否演變成缺貨漲價仍不確定」的限制。

5. Social Worker Daily 的健策、台達電、光寶操作紀錄屬筆記作者自己的資金調度，不能寫成主持人觀點或節目建議。

Topic map / 主題地圖

  • 市場心理：從半信半疑到全面相信、社群全面擁擠、資金開始找下一個族群。
  • 資金流：外部題材若受傷，台積電可能成為避風港；金融股與中小股接棒代表盤面風險偏好高。
  • 半導體零組件：PMIC、Power、DrMOS、SiC Super Junction MOSFET、PSU 功率密度、AI server 高階需求排擠。
  • 供應鏈政策：Nexperia、non-China、歐系/美系/台系供應鏈受惠可能。
  • 風險管理：社群冒名推標的、低市值標的不上節目、不要把筆記作者操作當成主持人建議。
  • 非股票觀察：林口房地產籌碼硬、小生意讓利資本模式、租金對創辦人的壓力。

Draft investment hypotheses / 投資假說草稿

1. 若 AI server 電源架構與 PSU 功率密度提升持續，部分功率元件與高階被動元件可能具備「用量增加 + 單價/毛利提升」的雙重效果；但需逐公司確認產品等級與驗證進度。

2. 若 non-China 供應鏈配置加速，台廠可能取得二供資格；但這是供應鏈缺口機會，不等同於長期 moat，需追蹤正式認證、出貨量與毛利率。

3. 若被動元件籌碼進一步混亂，短線可能轉為區間震盪與資金對決；基本面續強不必然等於追價勝率高。

4. 若台灣主動基金/ETF 的單一持股上限放寬並實際改變投信配置，台積電可能受益於制度性資金回流；但觸發點可能是其他熱門題材轉弱，而非單一公告立即反映。

5. 筆記作者操作日記可作為市場參與者行為觀察，不能作為交易訊號。
