Core takeaways / 核心結論

1. 本集投資主軸是 **AI 需求的外溢與排擠效應**：資金與需求不只集中在 GPU / 台積電，也開始影響功率元件、被動元件、封裝測試與後段產能。

2. 主持人對 **功率元件 / MOSFET 缺料與漲價** 語氣偏強，重點不是單一 ticker，而是「誰有產能、長約、集團資源、封裝能力」。

3. **封測 / OSAT** 被描述為議價權修復場景：部分 IC 設計客戶遇到下一季報價上調 30–50% 的說法，是本集最高價值但也最需要查證的主張。

4. **Meta NeoCloud / 算力出租** 的主持人解讀偏向「加倍下注 AI」而非退出競爭：用舊代或冗餘算力變現，降低 CFO 對 CAPEX 的壓力。

5. Secondary note 的個股操作（強茂、富鼎、減碼光寶科 / 南亞科）是筆記作者詮釋與個人操作紀錄，不應視為主持人建議。

Draft investment hypotheses / 投資假說草稿

1. **功率元件缺料假說**：若 MOSFET / 功率元件 lead time 拉長與漲價在 7 月後延續，具備產能、長約、集團支援與封裝能力的公司，毛利率與估值可能同步被重估。

2. **封測排擠效應假說**：若 AI 封裝 / 測試需求排擠成熟客戶，OSAT 報價與毛利率可能出現比市場預期更強的修復。

3. **Meta CAPEX 變現假說**：若 Meta 將內部 AI 基礎設施部分商品化，可能改變 NeoCloud 競爭格局，也讓市場重新評估其 AI 投資回收期。

4. **資金輪動假說**：當大盤重新創高，資金可能先回到已被市場認可的強勢族群；但若券商信用收緊或追價過熱，波動會加劇。
