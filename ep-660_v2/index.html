核心結論

  1. 高位市場仍有動能，但不是無風險。長線獲利者成本遠離，短線回檔容易被承接；但槓桿、處置與流動性限制會讓波動更不對稱。
  2. 散熱 / CoWoS 改料是風險案例。primary 與 secondary 都提到改散熱設計、CoWoS 短暫停線與復線；但具體 BOM、受影響公司與財務影響仍未驗證。
  3. 成熟製程轉強比較像提前拉貨與怕漲價，不宜直接寫成終端需求復甦。要追 VIS / UMC / GFS guidance 與 PC / notebook / smartphone forecast。
  4. 世界先進的訊號強於矽晶圓。source layer 對世界先進 / 成熟製程偏正向；環球晶則更像股價先行、基本面仍待確認。
  5. 被動元件是本集較明確的觀察主線。MLCC、鋁電容、鉭電容、電感、電阻都被提及；需以交期、漲價、訂單與毛利驗證。
  6. 記憶體 / CPU 類股偏風控語境。看不懂就不右側加碼，破 5 日線或長黑 K 才逐步處理；這不是追高建議。
