投資假說與追蹤計畫

1. 被動元件規格排擠

若 AI server / Vera Rubin 對高階 MLCC 與電容需求提升，日韓廠轉向高階料號會讓標準品供需轉緊，台系廠可能享有漲價與產品組合改善。

第一個驗證點：47uF / 22uF lead time、日廠與台廠毛利率。

2. 成本驅動漲價風險

若 TI / Rohm 漲價主要是原料或成本轉嫁，而終端仍只是補庫存，則股價反應可能短命。

第一個驗證點：終端 demand、庫存天數與毛利是否同步改善。

3. 世界先進特殊製程

VSMC 新加坡廠已有官方確認的 JV、投資規模與投產時程；若客戶承諾與產能利用率可見度高，特殊製程可能優於一般消費回補。

第一個驗證點：VSMC 建廠進度、NXP demand、VIS 法說。

4. 防守型廠務

市場震盪時，低 PE 且有訂單能見度的廠務股可能成為資金避風港；但必須查毛利、合約負債、客戶壓價與海外案場。

第一個驗證點：漢唐 / 聖暉 Q1 毛利、合約負債與客戶結構。

  • 被動元件升級：Murata / Samsung EM / 國巨等法說明確提及 AI server MLCC demand、lead time 拉長或 product mix 改善。
  • 鋁電容假說升級：Nichicon / Nippon Chemi-Con 財報顯示毛利改善，且台廠同步公布漲價或高階應用營收增加。
  • 成熟製程從補庫存變需求驅動：VIS / VSMC / NXP 公告長約、客戶承諾或產能利用率能見度。
  • 防守股可研究：漢唐 / 聖暉估值仍低，但毛利、訂單、現金流沒有惡化。
  • 市場風險升級：IC 設計、散熱、被動元件等強勢族群同步跌破趨勢，且大單買進無法帶動追價。
