股癌 EP668：Computex 2026、RTX Spark、地端 AI 與 AI PC 換機邏輯

NotebookLM primary + validated Social Worker Daily secondary note + supplementary triangulation + external verification。本文是研究筆記，不構成個人化投資建議。

一句話結論：EP668 的可投資研究價值在於「AI PC 從願景走向工作流 ROI」：NVDA 是最乾淨表達，ADBE 與 MU 升級為 priority research，QCOM/AMD/INTC/ARM 作競爭壓力監控；但短線仍要尊重槓桿清洗與估值風險。

核心結論

  • RTX Spark / unified memory 是本集主軸。 NVIDIA 官方資料支持 1 petaflop AI performance、最高 128GB unified memory、Blackwell RTX GPU 與 20-core Grace CPU 的 local agent PC 敘事。
  • Adobe 是反直覺研究題。 Adobe 與 NVIDIA 官方合作把 Photoshop、Premiere、Substance 3D 對 RTX Spark 最佳化，讓 ADBE 從「被 AI 取代」轉成「AI workflow moat 是否修復」。
  • MediaTek 是產業訊號，不是美股乾淨標的。 美股 portfolio 更適合用 NVDA 做主表達，用 QCOM/AMD/INTC/ARM 監控競爭格局。
  • MU 值得加入 active watch。 Local AI 與最高 128GB unified memory 支持容量升級邏輯；真正驗證點是 2026-06-24 Micron FYQ3 results。
  • Power Rack / LPU / ASP 100 倍不能直接建模。 這些是節目線索，仍需供應商 IR、價格函、lead-time 或官方資料。
  • 產業方向正向，但交易上不追高。 主持人同時提醒短線 brace for impact / 洗槓桿；研究方向與部位大小要分開。

投資假說草稿

NVDA — RTX Spark 是 AI PC 重新定義的 cleanest expression

若 RTX Spark 讓 local agents、creator workloads、gaming 整合成高 ASP 裝置，NVDA 可擴大 PC / creator ecosystem 的平台控制力。驗證看 OEM SKU、上市價格、review benchmarks、CUDA developer adoption 與毛利。

ADBE — AI workflow integration 可能比 AI replacement 更重要

如果 Photoshop / Premiere 的 RTX Spark optimization 讓專業工作流明顯加速，市場對 Adobe 被 AI 取代的敘事可能過度悲觀。驗證看 retention、enterprise pricing、AI features usage 與成長率修復。

MU — local AI memory capacity 是新增觀察，不是立即交易理由

最高 128GB unified memory 支持 high-capacity client DRAM 方向，但 MU 的股價仍主要受 HBM、DRAM cycle 與財報 guidance 影響。6/24 是下一個 evidence checkpoint。
