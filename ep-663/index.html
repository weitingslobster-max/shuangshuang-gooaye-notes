Core takeaways / 核心結論

1. **本集最高價值線索是被動元件供需與定價權回升**。主持人把 Panasonic 漲價信、現貨掃貨、低階料號產能紀律與高階 MLCC / 鋁電容需求放在同一條主線；這是研究優先度最高的投資假說，但仍需用正式漲價函、通路報價與各廠 capex/稼動率驗證。

2. **市場盤勢被描述為健康輪動，不是全面崩壞**。四月萬物齊漲後，資金開始往 MLCC、鋁電容等強勢族群集中；主持人同時提醒「強勢族群縮小」時要提高警覺。

3. **Google TPU / Broadcom / MediaTek 的技術代號與雙拼載板故事值得追，但證據層級較低**。節目提到 Zebrafish、Humu Fish、Sunfish 與大顆 Sunfish 雙拼載板；外部可確認的是 Google/Broadcom/Anthropic 對 TPU compute 的長約與需求強度，不能直接確認雙拼設計。

4. **老 AI 族群不是被長線否定，而是短線接刀風險高**。主持人把「毛利率可能只短暫一兩季」與 Nvidia 路線圖持續推進放在觀察框架內；secondary note 則加入「不破底再看」等技術門檻。

5. **Secondary note 的國巨、健策、群聯操作日記必須獨立隔離**。國巨加碼、健策停損、群聯賣出是筆記作者個人交易紀錄，不是主持人建議，也不是本報告的投資建議。
